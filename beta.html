Triggers

Default

  • Hover Off
  • Mouse Up

Active

  • Hover On
  • Hover Off
  • Mouse Down


Rules

Default

  • Hover On

    • Cursor changes to finger
    • Icon container becomes 10% darker in value
    • Modal appears instantly

  • Hover Off

    • Cursor changes to pointer
    • Like container box returns to original value
    • Modal disappears instantly

  • Mouse Up

    • Icon turns to black
    • User is taken to a different page corresponding tab

Active

  • Hover On

    • Cursor changes to finger
    • Modal appears instantly

  • Hover Off

    • Cursor changes to pointer
    • Modal disappears instantly

  • Mouse Down

    • Icon container becomes 10% darker in value

  • Mouse up

    • Page refreshes, server sends new list of emails, chat, etc.
