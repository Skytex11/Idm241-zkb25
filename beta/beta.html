Rules

Mail Icon: Default

  • Hover On:
    • Cursor changes to finger
    • Mail Icon container box becomes 10% darker in value
    • Mail Icon jumps up and down slightly for 1.5 seconds (animation)
    • 2 second delay, Side panel Modal appears
  • Hover Off:
    • Cursor changes to pointer
    • Mail container box returns to original value
    • Side panel Modal disappears
  • Mouse Down:
    • Mail container opacity appears 15% darker
  • Mouse Up:
    • Mail Icon turns black
    • Mail container opacity is 10 darker
    • User is taken to their mailbox
    • Data from server loads user’s emails

Mail Icon: Active

  • Hover On:
    • Cursor changes to finger
  • Hover Off:

  • Mouse Down:

  • Mouse Up:
    • Page refreshes

Loops

None

Modes

Mail Modal

The Mail Modal has 14 subsections and two of which are drop downs. All sections have a hover on and off trigger. When Hover On, the container turns 15% darker. When Hover Off, the container returns back to normal color.

Chat Modal

This Modal has four main components. In total, there are three drop down features between the four. Similar Hover On and Off function as the Mail Modal.
