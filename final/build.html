Active Button

Triggers

This button has different modes you can set to display users online status.

  • Hover On
  • Hover Off
  • Mouse Up
  • Mouse Down

Rules

  • Hover On:
    • Cursor changes to finger
    • Active button becomes 10% darker in value
  • Hover Off:
    • Cursor changes to finger
    • Active button returns to original value
  • Mouse Down:
    • Active button opacity appears 15% darker
  • Mouse Up:
    • Active button returns to original value
    • Drop down panel Modal appears

Feedback

Modes

When Button is default, user clicks on the button and a small drop down modal appears. Modal display 5 options, layout in a column: Automatic, Do not disturb, Set as away, Add status, Chat noticfication setting. All sections have a hover on and off trigger. When Hover On, the container turns 15% darker. When Hover Off, the container returns back to normal color.

When user clicks anywhere outside of the window, the modal with disappear.
