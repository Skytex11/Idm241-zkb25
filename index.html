RULES - Describe in detail all of the rules in existing microinteraction.

Default

  • Hover ON
    • Cursor changes to finger
    • Like container box becomes 10% darker in value
    • 2 second delay, "I like this" tagline appears
  • Hover OFF
    • Cursor changes to pointer
    • Like container box returns to orignal value
    • "I like this" tagline disappears
  • Mouse Down
    • Like container opacity appears 15% darker
  • Mouse Up
    • Like Icon with container does an animation which involves moving and changing colors. After short animation, Icon is black.
    • Database Like flag set to TRUE

Active

  • Hover ON
    • Cursor changes to finger
    • Like container box becomes 10% darker in value
    • 2 second delay, "Unlike" tagline appears
  • Hover OFF
    • Cursor changes to pointer
    • "Unlike" tagline disappears
  • Mouse Down
    • Like container opacity appears 15% darker
  • Mouse Up
    • Like Icon turns white
    • Database Like flag set to FALSE
