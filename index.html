RULES

Thumbs Up - Default

  • Hover ON - Cursor changes to finger
  • Hover ON - Like container box becomes 10% darker in value
  • 2 second delay, "I like this" tagline appears
  • Hover OFF - Cursor changes to pointer
  • Hover OFF - Like container box returns to orignal value
  • Mouse Down - Like container opacity appears 15% darker
  • Mouse Up- Like Icon with container does an animation which involves moving and changing colors. After short animation, Icon is black.
  • Mouse Up - Database Like flag set to TRUE

Thumbs Up - Active

  • Hover ON - Cursor changes to finger
  • Hover ON - Like container box becomes 10% darker in value
  • 2 second delay, "Unlike" tagline appears
  • Hover OFF - Cursor changes to pointer
  • Mouse Down - Like container box opacity appears 15% darker
  • Mouse Up - Like Icon turns white
  • Mouse Up - Database Like flag set to FALSE
